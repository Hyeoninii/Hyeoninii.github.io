# 2. `_config.yml`에 설정 추가

이제 내 블로그 페이지에 대한 데이터를 수집할 수 있도록 애널리틱스 계정과 연결 시켜주어야 한다.

다음과 같은 필드를 찾아서 `google id` 밑에 `컨테이너 ID`를 복사해서 넣어주어야 한다.

```yaml
analytics:
  google:
    id: G-XXXXXXXXXX
```

<br>

> 컨테이너 ID는 `G-`로 시작하는 애널리틱스 속성 페이지 또는 태그 관리자에서 확인 가능
